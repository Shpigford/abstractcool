Variables

Pass in any of the follow params to the URL to really spice it up...

width
Default: 500. The width of the image
height
Default: 500. The height of the image
cellSize
Default: 100. The size of the individual "cells". Generally, the smaller the cells, the slower the response time.
variance
Default: 1. Decimal value between 0 and 1 that specifies line/angle randomness.
